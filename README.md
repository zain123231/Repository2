# 🌍 GeoCLIP AI Locator: Ultimate Geographic Prediction System

![Project Status](https://img.shields.io/badge/Status-Active-success.svg)
![Python Version](https://img.shields.io/badge/Python-3.10-blue.svg)
![AI Framework](https://img.shields.io/badge/Framework-PyTorch-orange.svg)

مرحباً بك في مشروع **AI Geo-Locator**، وهو نظام ذكاء اصطناعي متطور مبني على نموذج **GeoCLIP**. 
هذا المشروع قادر على تحليل الصور بصرياً واستنتاج موقعها الجغرافي الدقيق على كوكب الأرض أو داخل دولة محددة (مثل العراق) باستخدام تقنيات مطابقة الصور (Image Embeddings) مع شبكة جغرافية عملاقة.

---

## 🚀 المميزات الرئيسية (Features)

1. **الذكاء الاصطناعي البصري (Computer Vision)**: يعتمد على بنية CLIP (ViT-L/14) لاستخراج البصمات الرياضية العميقة للصور.
2. **بحث جيومكاني فائق السرعة**: يستخدم مكتبة **FAISS** (من Facebook AI) للبحث في قاعدة بيانات تحتوي على قرابة 2 مليون موقع في أجزاء من الثانية.
3. **داتا سيت مخصصة لدولة العراق**: تم تحميل وبناء قاعدة بيانات تحتوي على أكثر من 55,000 معلم، قرية، وشارع داخل العراق لزيادة الدقة المحلية بشكل هائل.
4. **واجهة ويب تفاعلية (Web UI)**: تم بناء موقع متكامل وجميل يتيح للمستخدم العادي رفع الصور ومعرفة مواقعها بضغطة زر.

---

## 💻 اللغات والتقنيات المستخدمة (Tech Stack)

* **لغة البرمجة الأساسية**: Python 3
* **الذكاء الاصطناعي**: PyTorch (مع دعم CUDA لتسريع المعالجة بكرت الشاشة).
* **البنية الأساسية (Architecture)**: GeoCLIP (Image Encoder & Location Encoder).
* **معالجة البيانات الضخمة**: Pandas, NumPy.
* **البحث المتجهي (Vector Search)**: FAISS.
* **واجهة المستخدم (Frontend)**: Streamlit.
* **تحويل الصور**: PIL (Pillow), Torchvision.

## 🌐 كيفية تشغيل الموقع عبر المتصفح (How to Run Web UI)

واجهة المستخدم هي أسهل طريقة للتعامل مع الذكاء الاصطناعي. لتشغيل الموقع محلياً على جهازك، اتبع هذه الخطوات:

1. **افتح التيرمنل (PowerShell أو CMD)** وتأكد أنك داخل مجلد المشروع:
   ```bash
   cd C:\Users\Msi\Desktop\ai\geolocalization
   ```

2. **اكتب أمر التشغيل التالي**:
   ```bash
   python -m streamlit run app.py
   ```

3. **افتح المتصفح الخاص بك (Google Chrome, Edge, etc.)**:
   سيقوم التيرمنل بإعطائك رابطاً (عادة ما يكون `http://localhost:8501`).
   اكتب هذا الرابط في المتصفح وستفتح أمامك واجهة البرنامج.

4. **طريقة الاستخدام**:
   * قم برفع صورة (Upload Image).
   * يمكنك اختيار تفعيل خيار **Use Iraq-Only Database** إذا كنت متأكداً أن الصورة في العراق لتضييق نطاق البحث.
   * اضغط على زر **Predict Location**.
   * ستظهر لك بطاقات بالنتائج، انقر على **View on Google Maps** ليأخذك مباشرة إلى الموقع على الخريطة!

## 📊 تقرير الذكاء الاصطناعي (AI Report)

* تم الاعتماد على أوزان (Weights) مدربة مسبقاً لاستخراج خصائص الصور (Visual Features).
* واجهنا مشكلة في البداية حيث كان النظام الأصلي يخطئ في المعالم العربية لأنه تدرب في بيئة غربية.
* تم حل المشكلة هندسياً عبر تقنية **Region-specific Database Restriction**، حيث قمنا بتحجيم بحث خوارزمية L2 Distance داخل ملف `iraq_index.faiss` فقط، مما أجبر الشبكة العصبية على مطابقة البيئة الصحرواية والمعمارية بأقرب مثيلاتها حصراً داخل الحدود العراقية، وحققنا بذلك قفزة نوعية في الدقة.
* النظام الآن جاهز هندسياً لاستقبال عملية **Fine-Tuning** شاملة لأوزان الـ Location Encoder في المستقبل إذا أردنا تعليمه معالم جامعية محددة.
